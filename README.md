# FreeBite SF

FreeBite SF is a web app that helps people find free, cheap, and nearby food in San Francisco.

It shows food pantries, free meals, community fridges, student food resources, and budget-friendly food options in one place.

## Features

- Find free food resources
- Find cheap food nearby
- Sort by cheapest food
- Sort by closest location
- Filter by food type
- Search radius options
- Nearby restaurant fallback results
- Cheap item cards
- Directions links

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React
- OpenStreetMap / Overpass API
